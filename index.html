Introduction to Git

GIT is an open source version control system that is used to track changes made on our code.

Advantages of using GIT

  • Fast and Efficient
  • Distributed version control
  • Enhanced collaboration
  • Compactibility
  • Data integrity

Configuring GIT

git config --global user.name "quazeem idris folarinwa"

git config --global user.email "idrisquazeem46@gmail.com"

Basic GIT Command

  • git init : use to initialize a GIT repository
  • git add . : use to stage changes made to our code
  • git status : use to check the status of our code
  • git commit -m "commit message" : This is use to save the staged changes with a discriptive message
  • git remote add origin : This is use to connect our local repository to our remote repository on github
  • git push origin main : this is use to push your code from local repository to remote repository on github
